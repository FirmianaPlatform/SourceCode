Step 2 : Upload dataset

(1) Rule for file name :

Example 1: YOUR_FILE_NAME_F01_R01.raw

Example 2: YOUR_FILE_NAME_F02_R01.raw

Beginning "YOUR_FILE_NAME" is the origin file name or any name you like, "_F01" and "_R01" in tail mean "fraction number" and "repeat number".

(2) Upload path :

Experimenters of BPRC can upload data to the NAS path used before. In folder named "For Galaxy", there are existed folders, like "/Velos/80047/"
